## JAVASCRIPT OPERATORS

### 1. 연산자

##### (1) 산술 연산자(Arithmetic Operators)

숫자를 계산하는 데 사용되는 연산자.

- 더하기 : +

- 빼기 : -

- 곱하기 : *

- 거듭 제곱 : **

- 나누기 : /

- 나머지 : %

  

````````javascript
console.log(1 + 2);
````````

- 증가 : ++

전위 연산자(++x)는  x에 1을 더한 값을 반환하고, 후위 연산자(x++)는 x에 1을 더하기 전의 값을 반환함.

````````javascript
let a = 10; 
let b = a++; //a = 11, b = 10
let c = 10;
let d = ++c; //c = 11, d = 11
````````

- 감소 : --

++와 같음.

````javascript
let a = 10;
let b = a--; //a = 9, b = 10
let c = 10;
let d = --c; //c = 9, d = 9
````
